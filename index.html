Curso de jQuery Mobile ©2013

El curso de Desarrollo de Aplicaciones para Web y Móviles, ofrece el conocimiento práctico y las tecnologías necesarias para crear aplicaciones de alto impacto. La metodología del curso permite un proceso de aprendizaje inductivo y progresivo, guiando al estudiante para que maneje eficientemente los conocimientos adquiridos de manera práctica profundizando en las nuevas etiquetas y funciones de HTML5, CSS3 y jQuery y jQuery Mobile.

Objetivos:

● Permitir al aprendiz adquirir conocimientos y herramientas de última generación para el desarrollo de aplicaciones web y para dispositivos móviles.

● Acompañar al estudiante para que aplique los conocimientos de manera progresiva iniciando por conceptos básicos y llegando a un nivel avanzado al finalizar el curso
